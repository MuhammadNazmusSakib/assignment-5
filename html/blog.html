What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

How do you select an element from the DOM?

To select an element from the DOM, there are several methods provided by JavaScript. Here are the most common ones:

  • 1. getElementById() : selects only one element by its unique ID.
  • 2. getElementsByClassName() : return live collections (updates with DOM changes).
  • 3. getElementsByTagName() : return live collections (updates with DOM changes).
  • 4. querySelector() : use CSS selectors and are more flexible but are slower than the other methods.
  • 5. querySelectorAll() : use CSS selectors and are more flexible but are slower than the other methods.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.

Why Event Delegation is Useful:

Performance: Reduces memory usage and improves performance when many elements are involved, as fewer event listeners are added.
Dynamic Content: Works well with elements that are dynamically added to the DOM after the event listener is set up (e.g., when new child elements are added after an AJAX call).
Simplifies Code: Instead of attaching listeners to each child element, you manage all similar events from a single parent, keeping code simpler and more maintainable.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles in the DOM using various JavaScript methods we can use-

  • By using getAttribute(), setAttribute(), and removeAttribute() to manipulate an element's attributes.
  • By using the style property for inline style changes and classList for CSS class manipulation.
